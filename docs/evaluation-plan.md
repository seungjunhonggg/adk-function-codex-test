# 평가 계획 (v0)

## 목적
- 브리핑 서술 품질과 정확성을 자동 평가한다.
- 표/차트의 수치를 잘 인용하는지 검증한다.
- 라우팅 분류 정확도를 확인한다.

## 평가 방식
### 1) Direct Scoring (객관 기준)
- 기준: 정확성, 지시 준수, 간결성, 근거 인용
- 스케일: 1~5
- 증거 기반 채점(근거 텍스트 포함)

### 2) Pairwise Comparison (선호 기준)
- 기준: 가독성, 설명 흐름
- 위치 바이어스 완화: A/B 순서 바꿔 2회 평가
- 불일치 시 TIE 처리

## 루브릭(초안)
1. 정확성(1~5)
   - 표/차트 값과 서술이 일치하는가
2. 지시 준수(1~5)
   - 한글 라벨만 사용했는가
   - children 지표를 기본에서 숨겼는가
3. 간결성(1~5)
   - 2k~3k 토큰 목표를 크게 넘지 않는가
4. 설명 명료성(1~5)
   - 단계별 흐름이 이해되는가

## 바이어스 완화
- 길이 편향: 기준에 "불필요한 장문 감점" 포함
- 위치 편향: pairwise 순서 교환
- 자기 평가 편향: 생성 모델과 평가 모델 분리

## 산출물
- 평가 JSON
  - scores, reasons, final_summary
### 3) 라우팅 분류 평가
- 샘플 문장 20개로 simulation/casual 분류 정확도 측정
- ambiguous 문장은 별도 라벨로 모아 재질문 정책 확인
