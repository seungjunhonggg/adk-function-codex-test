# UI 스펙 (v0)

## 목표
- ChatGPT와 유사한 UX를 유지하되, MLCC 도메인에 맞는 차별화된 톤.
- 표/차트 중심 결과를 안정적으로 보여주는 구조.

## 비주얼 방향
- 톤: 산업적이되 정제된 연구 노트 느낌(차분 + 명료).
- 배경: 은은한 그레인/그리드 텍스처로 깊이감 추가.
- 색상: 웜 그레이 베이스 + 딥 틸 포인트. 보라색 계열은 사용하지 않음.

## 타이포그래피
- 헤딩: "Cormorant Garamond" (display, 영문)
- 본문/한글: "SUIT" 또는 "Spoqa Han Sans Neo"
- 숫자 강조: 탭уляр 숫자 사용(폰트 옵션 가능)

## 레이아웃
- 좌측: 세션/워크플로우 내비게이션
- 중앙: 대화/브리핑 본문
- 우측(옵션): 파라미터 요약/단계 진행 상태
- 입력창: 고정 하단, 단일 라인 + 확장 버튼

## 컴포넌트
- 브리핑 블록: 섹션 카드 + 표/차트 인라인
- 단계 진행 표시: 1-1~1-8 스텝퍼
- 표: 고정 헤더, 열 라벨 한글화
- 차트: 기본 막대, 요청 시 선/스캐터

## 모션
- 페이지 로드 시 섹션별 순차 등장(스태거)
- 표/차트는 가볍게 페이드 인

## 반응형
- 모바일: 좌측 내비게이션을 드로어로 전환
- 표는 가로 스크롤 허용

## 구현 메모
- 실제 구현 시 React + Tailwind + shadcn/ui 기반.
- web-artifacts-builder 스크립트로 아티팩트 생성 가능.
